# Faiz - Portfolio

Professional portfolio website for Faiz, a Video Editor & Graphic Designer specializing in YouTube content, reels, promos, and social media design.

## Features

- Modern, premium dark theme design
- Fully responsive across all devices
- Smooth scroll animations and transitions
- Interactive portfolio filtering
- Contact form with validation
- Optimized performance
- Accessible navigation

## Technologies

- HTML5
- CSS3 (Custom animations, gradients)
- Vanilla JavaScript (ES6+)
- Tailwind CSS (CDN)
- Iconify Icons

## Sections

- Hero with typing effect
- About with experience showcase
- Skills with animated progress bars
- Services overview
- Portfolio with category filtering
- Client testimonials
- Process timeline
- Contact form
- Social media links

## Performance

- Lazy loading images
- Optimized animations with requestAnimationFrame
- Minimal dependencies
- Efficient event handling
- CSS-based animations where possible

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
